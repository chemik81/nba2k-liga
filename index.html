🏀 Konfiguracja
Aby dane meczów były dostępne dla wszystkich, podłącz bazę danych JSONBin.io.
Dane zostaną zapisane tylko w Twojej przeglądarce.
1 Wejdź na jsonbin.io i załóż darmowe konto
2 Kliknij API Keys → skopiuj Master Key
3 Kliknij Bins → New Bin, wklej {"matches":[],"aliases":{}} i zapisz
4 Skopiuj Bin ID z URL
5 Cloudflare Worker przechowuje klucz Anthropic bezpiecznie — nie musisz go tu wpisywać.
Klucz Gemini przechowywany bezpiecznie w Cloudflare Worker jako secret GEMINI_API_KEY.
Zostaw puste aby nie zmieniać. Hasło jest przechowywane lokalnie jako hash.
